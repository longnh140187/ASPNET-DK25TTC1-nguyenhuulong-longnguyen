## 4. Tổng kết dự án (hoàn thành)

### Chức năng người dùng
- Trang chủ: công thức nổi bật, món yêu thích, blog mới
- Danh sách công thức: tìm kiếm, lọc danh mục, độ khó, thời gian nấu, phân trang
- Chi tiết công thức: nguyên liệu, các bước nấu, thông tin dinh dưỡng
- Danh sách và chi tiết blog

### Chức năng Admin
- Đăng nhập / đăng xuất
- Dashboard quản trị
- CRUD: Danh mục, Công thức, Blog, Người dùng
- Upload ảnh thumbnail

### Hạ tầng
- ASP.NET Core 8 MVC + EF Core + MySQL (Pomelo)
- Docker Compose (`make up` → http://localhost:5080)
- Migration + seed dữ liệu demo tự động
